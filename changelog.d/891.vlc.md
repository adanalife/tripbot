**New `tripbot_current_state` gauge exposes which US state the dashcam playhead is currently in.** A labeled gauge (`tripbot_current_state{state="MO"} 1`) sourced from `pkg/video.Current().State` at the existing video-transition site. On each transition the new state's series is set to `1` and the previously-active series cleared to `0`, so exactly one series reads `1` at a time — no stale `=1` series linger.
